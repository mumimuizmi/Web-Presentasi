CARA MEMBANGUN PROYEK WEBSITE :

LANGKAH 1
Buka browser dan akses halaman login WordPress, kemudian masukkan username dan password untuk masuk ke dashboard WordPress.
LANGKAH 2
Setelah berhasil masuk, akan tampil Dashboard WordPress yang digunakan untuk mengelola website
LANGKAH 3
Pilih menu Appearance > Themes, kemudian klik Add New Theme untuk menambahkan tema website.
LANGKAH 4
Cari tema Astra pada kolom pencarian, kemudian klik Install dan Activate.
LANGKAH 5
Setelah tema Astra aktif, pilih menu Plugins > Add New Plugin untuk menambahkan plugin Elementor.
LANGKAH 6
Cari plugin Elementor Website Builder, kemudian klik Install Now dan Activate.
LANGKAH 7
Setelah Elementor aktif, pilih menu Pages > Add New Page untuk membuat halaman baru pada website.
LANGKAH 8
Beri nama halaman sesuai kebutuhan, kemudian klik tombol Edit with Elementor.
LANGKAH 9
Mulai mendesain halaman website dengan menambahkan section, heading, teks, tombol, dan elemen lainnya menggunakan Elementor.
LANGKAH 10
Tambahkan gambar yang telah disiapkan sebagai banner, galeri, atau konten pendukung website.
LANGKAH 11
Atur tata letak, warna, ukuran font, dan elemen lainnya agar tampilan website lebih menarik dan responsif.
LANGKAH 12
Setelah desain selesai, klik tombol Terbitkan atau Update untuk menyimpan perubahan pada website.
LANGKAH 13
Lakukan pengecekan tampilan website pada browser untuk memastikan seluruh halaman dan fitur berjalan dengan baik
LANGKAH 14
Hasil website
